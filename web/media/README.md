# Reference Motion Assets

The frontend uses lightweight CSS/HTML motion UI, not raw MP4 playback.

Use this folder only for local reference files while designing motion concepts. Video files are ignored by git so large generated assets are not committed accidentally.

Current motion concepts represented in CSS:

- Farmer + AI assistant hero.
- RAG pipeline flow.
- Dynamic live-data router.
- Sticky-note retrieval.
- Source/citation stack.
